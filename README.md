# Adobe Stock AI CSV Generator

## Overview
Web-based automation tool designed to streamline stock image publishing workflows.

The application integrates OpenAI API to automatically generate:
- Titles
- Keywords
- Categories

Metadata is structured into a CSV file ready for bulk upload.

## Features
- Upload multiple AI-generated images
- Generate structured metadata
- Export results as output.csv
- Designed for efficiency and workflow optimization

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js (API layer)
- OpenAI API
- Netlify (deployment)

## Purpose
This project was created to automate manual metadata preparation for stock publishing platforms and improve productivity.
